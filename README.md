# NaveedArts - Handmade Oil & Acrylic Paintings E-Commerce

Premium handmade paintings platform with admin dashboard.

## Features

✅ Admin Dashboard (add/edit/delete paintings without coding)
✅ Firebase Backend
✅ Luxury Design
✅ WhatsApp Integration
✅ Global Shipping

## Quick Start

1. Install dependencies: `npm install`
2. Setup Firebase config in `.env.local`
3. Run: `npm run dev`
4. Visit: http://localhost:3000/admin

See SETUP_GUIDE.md for detailed instructions.
